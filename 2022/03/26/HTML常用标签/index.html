4.6 图像标签和路径

1.图像标签

​ 在HTML标签中，<img>标签用于定义HTML页面中的图像

​ src是<img>标签的必须属性，它用于指定图像文件的路径和文件名

​ 可以理解为属于这个图像标签的特性

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本。图像不能显示的文字
title 文本 提示文本。鼠标放在图像上可以显示文字
width 像素 设置图像的宽度
height 像素 设置图像的高度
border 像素 设置图像的边框粗细

​ 具体效果可点击这里查看

2.路径

#### (1)目录文件夹和根目录

​ 目录文件夹:普通的文件夹，存放了我们制作页面所需要的相关素材，用来管理他们的文件就叫做目录文件夹。

​ 根目录:打开目录文件夹的第一页就是根目录。

(2) vscode中打开目录文件夹

​ 文件 ——> 打开文件夹即可

(3)路径之相对路径

  1. 相对路径 :以引用文件所在位置为参考基础，而建立出的目录路径。

    ​ 这里所使用样例是以图片相对于HTML页面的位置。

    | 相对路径分类 | 符号 | 说明 |
    | ———— | —- | ——————————————————- |
    | 同一级路径 | | 图像文件位于HTML文件同一级 如<img src="baidu.gif"> |
    | 下一级路径 | / | 图像文件位于HTML文件同一级 如<img src="baidu.gif"/> |
    | 上一路径 | ../ | 图像文件位于HTML文件上一级 如..<img src="baidu.gif"/> |

4.7 超链接标签

### 1.链接的语法格式

​ 标准格式：

​ <a href="跳转目标" target="目标窗口弹出方式”> 文本或图像 </a>

​ 两个属性的作用如下：

属性 作用
href 用于指定链接目标的URL地址，(必须属性) 当为标签应用href属性时，他就具有了超链接功能。
target 用于指定链接页面的打开方式，其中_self为默认值,_blank为在新窗口中打开的方式。

2.链接分类：

​ 1.外部链接：例如< a href="http://www/baidu.com"> 百度</a>

​ 2.内部链接：网站内部页面之间的相互链接. 直接链接内部页面名称即可，例如

​ <a href="index.html">首页</a>
